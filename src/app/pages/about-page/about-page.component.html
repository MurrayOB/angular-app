About This Site

This site was built with Angular TS, Bootstrap, Charts.js and Unicons for developer demonstrative purposes. Feel free to play around with this simple site, creating fake products, removing products, viewing the statistical graph change based on products and/or edit your username. A Rest API was built for the site but is not in use for the demonstration.
